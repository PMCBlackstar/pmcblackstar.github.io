Claim

The box has some type of bucket system or easy way to hold a lot of water and realse it randomly..

Evidence

After, when then poured 200 ml twice; first output was around half of 100ml and around ¾ of 200 for the second trial. Then right after we poured 400ml of water and we got an output of zero for the first 400ml and then for the second 400ml we poured in around 270 came out.

Reasoning 1

The trials we did produced the same randomization of outputs as the original box and replicated the further trials. The key concept that we proved to be correct was the use of a bucket system.

Reasoning 2

The box produced similar results when we first ran trials which tells us that we were doing something right.

Reasoning 3

If there wasn’t some sort of water bucket system in the box then it would be difficulty for the output to be zero after putting in a little bit of water.
